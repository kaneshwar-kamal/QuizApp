# QuizApp

A basic **Flutter Quiz Application** built to practice Flutter UI development and app logic using **Dart**.  
This project is a multi-platform Flutter setup, so it can be run on **Android, iOS, Web, Windows, macOS, and Linux** (depending on your environment).

## Project Explanation
The goal of this project is to provide a simple quiz experience in a clean Flutter app structure.  
A user typically goes through a set of questions, selects answers, and the app calculates a score (or result) based on the selected options.

Even though the app logic is written in **Dart**, the repository contains additional languages like **C/C++/CMake/Swift/HTML** because Flutter generates platform-specific project folders for Android, iOS, desktop, and web.

## What you can learn from this project
- Flutter project structure (`lib/`, `assets/`, platform folders)
- Building UI using Flutter widgets
- Managing state (basic state handling for quiz questions and score)
- Navigation between screens (if implemented)
- Using assets (images/fonts) (if included)

## Tech Stack
- **Flutter**
- **Dart**

## Folder Structure (common Flutter layout)
- `lib/` — Main application source code (UI + quiz logic)
- `assets/` — Images, fonts, or data files (if used)
- `test/` — Unit/widget tests
- `android/`, `ios/`, `web/`, `windows/`, `macos/`, `linux/` — Platform-specific build files generated by Flutter

## How to Run the Project

### Prerequisites
- Flutter SDK installed
- Android Studio / VS Code (recommended)

### Setup & Run
1. Clone the repository:
   ```bash
   git clone https://github.com/kaneshwar-kamal/QuizApp.git
   cd QuizApp
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```
